# 📊 Outstanding Balance Analysis Dashboard

## 🚀 Overview
This project presents an interactive Tableau dashboard developed to analyze financial data related to clients, products, interest rates, and outstanding balances. The primary objective is to transform raw data into meaningful, actionable insights that support data-driven decision-making in a business environment.

The dashboard leverages advanced data visualizations to explore the distribution of outstanding balances across interest rates, identify correlations between key variables using trend analysis, and track top-performing clients over time. Interactive features such as dynamic filters enable users to drill down into specific time periods and client segments, enhancing analytical flexibility.

Overall, this project demonstrates strong capabilities in data visualization, dashboard design, and business insight generation, showcasing the ability to convert complex datasets into intuitive and impactful visual stories.

---

## 🎯 Objectives
- Analyze the distribution of outstanding balances across different interest rates  
- Identify correlations between interest rates and outstanding balances  
- Highlight the top 5 clients with the highest outstanding balances over time  
- Enable dynamic data exploration using interactive filters  

---

## 🛠️ Tools & Technologies
- **Tableau** – Data visualization and dashboard development  
- **Microsoft Excel** – Data preparation and preprocessing  

---

## 📊 Key Features

### 🔹 Distribution Analysis
A bar chart visualizes how outstanding balances are distributed across various interest rates, helping identify concentration patterns and financial exposure.

### 🔹 Correlation Analysis
A scatter plot with a trend line is used to examine the relationship between interest rates and outstanding balances. The visualization reveals a **positive correlation**, indicating that higher interest rates are associated with higher outstanding balances.

### 🔹 Top Clients Over Time
A time-series line chart highlights the **top 5 clients** based on outstanding balances, enabling users to track performance trends and identify key contributors over time.

### 🔹 Interactive Dashboard
- Dynamic **Reporting Date filter** for time-based analysis  
- **Client selection filter** for focused insights  
- User-friendly layout for seamless data exploration  

---

## 📈 Key Insights
- Outstanding balances tend to increase with higher interest rates  
- A consistent upward trend is observed among top clients over time  
- A small group of clients contributes significantly to total outstanding balances
